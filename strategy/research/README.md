# 🔍 Web Research Intelligence

This directory contains automated market intelligence briefs generated by the **Web Research Agent**.

## How It Works

The `web-research-agent.yml` workflow runs every 12 hours and performs real web searches across:

| Source | What It Captures |
|--------|-----------------|
| **Google News RSS** | Industry news, funding rounds, product launches, market shifts |
| **HackerNews** | Developer sentiment, Show HN launches, technical discussions |
| **Reddit** | User pain points, product reviews, feature requests, complaints |
| **GitHub** | OSS competition, trending repositories, tech evolution |
| **ProductHunt** | New AI/audio product launches |

## Output

Each run produces a dated intelligence brief: `YYYY-MM-DD.md`

Briefs contain:
- **Market Signals** — news, funding, launches
- **Competitor Activity** — product updates, pricing changes, positioning
- **User Signals** — real quotes from users about pain points and needs
- **OSS Trends** — rising open-source alternatives
- **Actionable Opportunities** — linked to GitHub issues

## Usage

Other agents (CPO, CMO, Standup Facilitator) read the latest brief to ground their analysis in real-world evidence rather than assumptions.
